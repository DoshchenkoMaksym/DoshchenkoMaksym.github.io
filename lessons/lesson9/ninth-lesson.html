 Псевдоклас находит один или более элементов с заданным тегом, основываясь на их позиции среди группы соседних элементов. В скобках так же можно вставлять значения - odd (нечётный) even(чётный).
  • Псевдокласс :disabled
CSS псевдокласс :disabled находит любой отключенный элемент. input:disabled Выберет все отключенные поля ввода
  • Псевдокласс :enabled
CSS псевдокласс :enabled находит любой включенный элемент.
  • Псевдокласс :required
CSS псевдокласс :required находит любые < input> элементы, имеющие атрибут required
  • Псевдокласс :optional
CSS псевдокласс :optional находит любые < input> элементы, у которых не установлен атрибут required
  • Псевдокласс :invalid
CSS псевдокласс :invalid находит любые < input> или < form> элементы, контент которых не проходит валидацию, в соответствии с типом поля.
  • Псевдокласс :valid
CSS псевдокласс :valid находит любые < input> или < form> элементы, контент которых валиден, в соответствии с типом поля.
  • Псевдокласс :empty
псевдокласс :empty находит любой элемент, у которого нет потомков. Учитываются элементы и текст (включая пробелы). Комментарии не повлияют на то, что элемент будет рассматриваться как не пустой.
  • Псевдокласс :focus
CSS псевдокласс :focus применяется, когда элемент (такой как input формы) получает фокус. Обычно он активируется при клике мышью пользователем или при выборе элемента с использованием клавиши "tab" на клавиатуре.
  • Псевдокласс :not()
Отрицательный CSS псевдокласс (en-US), :not(X) - функция, принимающая простой селектор X в качестве аргумента. Он находит элементы, не соответствующие селектору. X не должен содержать других отрицательных селекторов.

cursor: pointer

  • Псевдокласс :active
:active
Нажми на меня
Псевдокласс :active соответствует элементу в момент, когда он активируется пользователем. Он позволяет странице среагировать, когда элемент активируется. Взаимодействие элемента с мышью - это, как правило, время между нажатием и отпусканием пользователем кнопки мыши.
