Макеты
Имя Пример Описание
  • Команда git branch
Команда при запуске которой без параметров, получаем простой список имеющихся веток
  • Команда git clone
Команда осуществляет клонирование репозитория.
  • Команда git fetch
Команда связывается с удалённым репозиторием и забирает из него все изменения, которых у вас пока нет и сохраняет их локально.
  • Команда git pull
Команда работает как комбинация команд git fetch и git merge , т. е. Git вначале забирает изменения из указанного удалённого репозитория, а затем пытается слить их с текущей веткой.
    Команда git reset
  • Обычный
  • Жёсткий режим --hard
  • Обычный режим - удаляет комитты, но не сами файлы.
  • Удаляет комитты, и сбрасывает файлы
  • Команда git revert
    Команда – безопасный способ отменить операцию без потери истории коммитов. Команда отменяет действия прошлых коммитов, создавая новый, содержащий все отменённые изменения.
