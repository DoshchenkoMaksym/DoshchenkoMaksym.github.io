Макеты
Имя Пример Описание
  • Команда pwd
Команда pwd выводит текущую рабочую директорию.
  • Команда cd
Команда CD используется для просмотра или изменения пути текущего каталога .
  • Команда git init
 Для создания нового репозитория используется команда git init. Команду git init выполняют только один раз для первоначальной настройки нового репозитория. Выполнение команды приведет к созданию нового подкаталога .git в вашем рабочем каталоге. Кроме того, будет создана новая главная ветка.
  • Команда git add
 Команда git add добавляет содержимое рабочего каталога в индекс (staging area) для последующего коммита.
  • Файл .gitignore
 .gitignore нужен для скрытия файлов и папок от системы контроля версий Git. Обычно скрывают конфигурационные файлы (особенно с паролями), временные файли и папки.
  • Команда git commit
 Команда git commit берёт все данные, добавленные в индекс с помощью git add , и сохраняет их слепок во внутренней базе данных, а затем сдвигает указатель текущей ветки на этот слепок.
  • Команда git config --global user.name ""
  • git config --global user.email
Команды указывают пописывают имя и адрес электронной почты. Каждый коммит в Git содержит эту информацию, и она включена в коммиты, которые передаются, и не может быть далее изменена
  • Команда git log
 Позволяет посмотреть, что было сделано — историю коммитов.
  • Команда git diff
 Команда которая инициирует функцию сравнения источников данных
  • Команда git checkout -b
 Команда создает ветку и сразу переключается на нее
  • Команда git checkout
 Команда которая позволяет переключатся между ветками
  • Команда git merge
 Команда git merge отвечает за объединение ветвлений в истории Git и позволяет выполнять слияние отдельных процессов разработки, созданных с помощью команды git branch, в единую ветку.
  • Команда rm
 Команда rm используется для удаления файлов и каталогов в UNIX-подобной операционной системе.
  • Команда ls
 Команда ls используется в командной оболочке Linux для вывода содержимого каталогов и информации о файлах.
  • Команда cat
 Команда cat считывает данные из файлов и выводит их содержимое.
  • Команда git remote add origin
 Позволяет связать данный локальный репозиторий с удалённым, причем назначить этот удалённый репозиторий основным (origin)
  • Команда git push
 Команда позволяет отправить данные в удалённый репозиторий.
