Макеты
Имя Пример Описание
  • Тег < form> < /form>
  • Тег < form> устанавливает форму на веб-странице. Форма предназначена для обмена данными между пользователем и сервером.
Запросы в forme:
  • get
  • post
  • GET - Запрашивает данные из указанного ресурса. Осуществляет GET-запрос к серверу, то есть все данные запроса передаются в строке запроса.
  • Метод запроса POST предназначен для запроса/ В отличие от GET-запросов данные POST-запросов передаются не в строке запроса, а в его теле.
Атрибуты в form
  • action="url"
  • method="post/get"
  • enctype="multipart/form-dat"
  • Адрес программы или документа, который обрабатывает данные формы.
  • Метод протокола HTTP.
  • Данные не кодируются. Это значение применяется при отправке файлов.
  • Поле ввода текста type="text"
  • Текстовое поле. Предназначено для ввода символов с помощью клавиатуры.
  • Атрибут для input - value type="text" value="Пример"
  • Атрибут отображает начальное значение в строке.
  • Атрибут для input - placeholder, type="text" placeholder="ПРИМЕР"
  • Атрибут отображает подсказку в строке. Исчезает если начать вводить текст.
  • Текстовое поле ввода пароля type="password"
  • Обычное текстовое поле, но отличается от него тем, что все символы показываются звездочками. Предназначено для того, чтобы никто не подглядел вводимый пароль.
Поле ввода чисел type="number"
Имеет атрибуты
  • man max
  • step
min="0" max="50" step="10"
  • Поле ввода чисел. С возможностью устанавливать диапазон и шаг.
Поле ввода чисел type="range"
Имеет атрибуты
  • man max
  • step
min="0" max="50" step="10"
  • Поле ввода чисел в виде ползунка в указанном диапазоне.
  • Текстовое поле ввода email, type="email"
  • Предназначено для адресов электронной почты.
  • Текстовое поле ввода телефона, type="tel"
  • Предназначено для телефонных номеров.
  • Виджет для выбора цвета. type="color"
  • Предназначено для выбора цвета.
  • Флажки. type="checkbox"
  • Позволяют выбрать более одного варианта из предложенных.
  • Флажки. type="checkbox" с атрибутом checked
  • Позволяют выбрать более одного варианта из предложенных.checked - сразу выбран вариант

  • Тег Select
  • Тег < select> позволяет создать элемент интерфейса в виде раскрывающегося списка, а также список с одним или множественным выбором
  • Тег Select c тегом optgroup
  • Атрибут внутри optgroup -label(название группы)
  • Тег < optgroup> представляет собой контейнер, внутри которого располагаются теги < option> объединенные в одну группу.
  • Тег Select c тегом optgroup
  • Атрибут внутри optgroup -label(название группы)
  • Атрибут внутри optgroup -multiple(можно выбрать несколько вариантов)
  • Тег внутри < optgroup> дает название группы.
  • Тег внутри < select> дает выбрать несколько вариантов ответа.
  • Тег Select с атрибутом selected
  • Делает текущий пункт списка выделенным.
  • Тег textarea
  • Поле < textarea> представляет собой элемент формы для создания области, в которую можно вводить несколько строк текста.
