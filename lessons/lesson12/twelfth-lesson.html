Макеты
Имя Пример Описание
  • tranform: rotate (угол)
transform: rotate(45deg);
 Функция CSS rotate () определяет преобразование, которое перемещает элемент вокруг неподвижной точки
  • tranform: scale (значение)
transform: scale(1.5);
 Масштабирует элементы, делая их больше или меньше. Значения от 0 до 1 уменьшают элемент.
  • tranform: skew (значение)
transform: skew(30deg);
 Используется для деформирования (искажения) сторон элемента относительно координатных осей. Если указано одно значение, второе будет определено браузером автоматически.
  • tranform: translate (значение)
transform: translate(50px, 10px);
 Сдвигает элемент на новое место, перемещая относительно обычного положения вправо и вниз, используя координаты X и Y, не затрагивая при этом соседние элементы. Если нужно сдвинуть элемент влево или вверх, то нужно использовать отрицательные значения.
  • tranform-origin: Координаты
transform-origin:top right ; transform: rotate(45deg);
 Свойство позволяет сместить центр трансформации, относительно которого происходит изменение положения/размера/формы элемента.

  • flat
  • preserve-3d
  • preserve-3d Показывает, что дочерний элемент должен быть спозиционирован в 3D-пространстве.
  • flat Показывает, что дочерний элемент лежит в той же плоскости, что и родительский.
  • transition-property: all
  • Устанавливает имя стилевого свойства, значение которого будет отслеживаться для создания эффекта перехода.
  • transition-duration: 5s
  • Свойство предоставляет способ контролировать скорость анимации, при изменении CSS свойств.
  • transition-timing-function: ease
  • Устанавливает, насколько быстро должно изменяться значение стилевого свойство для которого применяется эффект перехода.
  • transition-delay: 1s
  • Свойство transition-delay устанавливает время ожидания перед запуском эффекта перехода.
  • animation-name
  • CSS свойство animation-name задаёт список анимаций, чтобы применить к элементу. Каждое имя является правилом @keyframes (en-US), которое задаёт значения свойств для последовательности анимации.
  • animation-duration
  • Свойство animation-duration устанавливает длительность анимации во времени за один цикл.
  • animation-delay
  • Свойство определяет время задержки перед стартом анимации.
  • animation-iteration-count
  • CSS свойство animation-iteration-count определяет сколько раз будет проигрываться анимационный цикл, перед тем как остановиться.
  • animation-timing-function
  • CSS свойство animation-timing-function задаёт как происходит анимация в течении длительности каждого цикла.
  • animation-direction
  • CSS-свойство animation-direction определяет, должна ли анимация воспроизводиться вперёд, назад или переменно вперёд и назад.
  • animation-play-state
  • Свойство animation-play-state определяет состояние анимации, паузы или проигрыша. Это можно использовать, чтобы определить текущее состояние анимации, например, в скриптах.
  • animation-fill-mode
  • CSS свойство animation-fill-mode определяет, как нужно применять стили к объекту анимации до и после её выполнения.
