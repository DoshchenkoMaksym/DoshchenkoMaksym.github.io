  1. em
  2. vw
  3. vh
  1. Пример 2em
  2. Пример 2vw
  3. Пример 2vh
  1. 1em=100%. Значение в em высчитывается относительно текущего шрифта
  2. Процент от ширины экрана
  3. Процент от высоты экрана

 Способы задачи значений:
  1. По названию(константа цвета)
  2. По шестнадцатеричному значению
  3. По шестнадцатеричному значению(сокращенная форма). Каждый символ следует удваивать.
  4. По десятеричному значению
  5. По десятеричному значению с альфа каналом(определяет непрозрачность элемента)
  6. Установка цвета по цветовому кругу Итана
  7. Установка цвета по цветовому кругу Итана с альфа каналом(определяет непрозрачность элемента)
